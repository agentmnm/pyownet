.. topic:: Abstract

   pyownet is a pure python package that allows to access OWFS_ 
   via the `owserver network protocol`_ , in short *ownet*.

   .. _OWFS: http://owfs.org/
   .. _owserver network protocol:
      http://owfs.org/index.php?page=owserver-protocol
